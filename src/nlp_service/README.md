# nlp-service

Сервис для обработки текста.

В рамках данного проекта используется для обработки сообщений из Telegram-чата для:
- извлечения из них адреса,
- классификации описанной в них проблемы,
- форматирования извлеченного адреса,
- перевода его в координаты объекта.

Для контроллеров `/processing` используются Natasha и страндартные средства Python.

> ### `GET /processing/address`
> Возвращает адресс, извлеченный из текста.
>
> Параметры:
> - `message` (string) - текст.
>
> Ответ:
> - (string) - адрес.


> ### `GET /processing/problem`
> Возвращает класс проблемы, извлеченный из текста.
>
> Параметры:
> - `message` (string) - текст.
>
> Ответ:
> - (string) - класс проблемы.
>
> Доступные классы:
> 1. `Водоснабжение`
> 2. `Теплоснабжение`
> 3. `Электроснабжение`
> 4. `Дорожное снабжение`

Для контроллера `/geocoding` используются средства YandexMaps API.

> ### `GET /geocoding`
> Возвращает полный адрес и координаты, определенные по входному адресу.
>
> Параметры:
> - `address` (string) - адрес.
>
> Ответ:
> - [ (string), - полный форматированный адрес.
> - (string) ] - строка с координатами.
